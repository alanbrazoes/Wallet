<h1>Wallet</h1>

<p>Wallet é uma carteira digital que faz o acompanhamento de seus gastos e convertendo para uma moeda de sua escolha.</p>
<h3>Stacks</h3>
Javascript
ReactJS
Redux
<h3>Como rodar em sua máquina:</h3>

<code>npm install</code>
<br>
<code>npm start</code>
